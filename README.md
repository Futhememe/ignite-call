# Ignite-Call

Ignite-Call is a private project with version 0.1.0.

## Scripts

The following scripts are available for use:

- `dev`: runs the Next dev server
- `build`: builds the application
- `start`: starts the application
- `lint`: runs the linter

## Dependencies
